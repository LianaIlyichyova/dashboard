# Dashboard Project

This is a modern dashboard application built with **React 18**, **TypeScript**, and **Ant Design**, powered by **Vite** for fast development and optimized builds. It features responsive layouts, charts, and light/dark mode support.

## Features

- **React 18 + TypeScript** for type-safe, modern frontend development.
- **Vite** as the build tool for lightning-fast development and HMR.
- **Ant Design (antd)** for ready-to-use UI components.
- **Chart.js** for rendering charts (Bar, Line, Doughnut).
- **Light and Dark Mode**: Users can switch themes from the **Settings** page.
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices.
- **Persistent Settings**: Theme and other settings are saved in `localStorage`.
- **Reusable Components**: Cards, charts, navigation menu, and more.

## Available Scripts

Run the project locally:

```bash
# Install dependencies
npm install

# Start development server (Vite)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
